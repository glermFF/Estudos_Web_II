# 📚 Projeto Full Stack - Spring + React

Este repositório contém um projeto full stack desenvolvido durante as aulas de Sistemas Web II, com a proposta de aprender na pratica o desenvolvimento de aplicações modernas com **backend em Spring**, **frontend em React** e **infraestrutura em Docker** contendo containers para o backend e frontend.

O projeto é evoluído de forma incremental a cada aula, com a adição de novas funcionalidades, correções e melhorias contínuas.

## 💡 Objetivo do Projeto

* Aprender conceitos de desenvolvimento **full stack**.

* Praticar a integração entre backend e frontend via API REST.

* Explorar o uso de containers com Docker para facilitar o ambiente de desenvolvimento e a portabilidade da aplicação.

* Evoluir o projeto com novas funcionalidades a cada aula.

## 🛠️ Tecnologias Utilizadas

### Backend:

* Java 17+
* Spring Data JPA
* Spring Web
* PostgreSQL
* Maven

### Frontend:

* TypeScript
* React Router DOM
* Axios
* Bootstrap / Tailwind CSS

## 🚀 Estrutura do Projeto

```
/backend       -> Projeto Spring Boot
/frontend      -> Projeto React
/docker-compose.yml -> Orquestração dos containers
```

Cada parte é desenvolvida e versionada no mesmo repositório, em pastas separadas.

## 📝 Funcionalidades Implementadas

* [x] Configuração inicial dos projetos Backend e Frontend
* [x] Criação das imagens Docker para cada componente
* [x] Orquestração com Docker Compose
* [ ] APIs REST expostas pelo Backend
* [ ] Consumo das APIs no Frontend
* [ ] Interface de listagem e cadastro de dados

## 📦 Como Executar o Projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/glermff/Estudos_Web_II.git
```

### Executar Docker Compose

```bash
docker-compose up --build
```
